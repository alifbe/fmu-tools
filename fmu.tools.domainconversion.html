class fmu.tools.domainconversion.dconvert.DomainConversion(cube_template, depth_surfaces, time_surfaces, use_interval_velocity=False, time_to_depth=True)[source]

Bases: object

Domain conversion, tailored for fmu-sim2seis but also works as standalone.

The principle is to use matching sets of time and depth surfaces to create a velocity (or slowness) cube, and use that further to domain convert either seismics or surfaces.
